  • startUpdating(every:sendingUpdatesTo:) Extension method

    Start performing periodic updates, updating every updateInterval seconds.

    The passed closure will be added to the array of closures that will be called when any of the source’s values are updated. The closure will be called with all values, but not all the values will neccessary be new.

    The returned object must be retained; the lifecycle of the listener is tied to the object. If the object is deallocated the listener will be destroyed.
