  • TimerBasedDataSource

    The base for a DataSource that uses a Timer to provide periodic updates. Subclasses should implement the missing properties and methods of ManuallyUpdateableDataSource. By adding the missing properties and methods for ManuallyUpdateableDataSource and adding conformance to CustomisableUpdateIntervalDataSource the methods required by CustomisableUpdateIntervalDataSource will be provided for free via an extention
